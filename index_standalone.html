Nota sobre Políticas de Segurança CORS:

Ao abrir o ficheiro index.html diretamente clicando nele no seu computador (protocolo file://), o seu navegador de Internet bloqueia a leitura do ficheiro de dados perguntas.json por motivos de segurança.

Para solucionar este problema:

  • Abra este projeto usando a extensão Live Server no VS Code.
  • Ou abra uma consola no diretório do projeto e execute: python -m http.server 8000, e aceda a http://localhost:8000.
